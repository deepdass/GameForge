GameForge is where teenagers build real games, ship them publicly, earn prizes, and experience the full game development cycle -- from initial commit to player feedback.

How it works

Pick a Constraint

Pick a set of creative constraint to build your game around. This limits scope and fuels creativity.

Build & Log

Build your game and post devlogs documenting problems, decisions, and progress and track your Code time via Hackatime, artlogs via Lapse.

Ship It

Ship it on itch.io with open source code on GitHub and earn Sparks based on quality and effort.

Get Feedback

Players give public, categorized feedback. Top voted feedback becomes your next challenge.

Iterate & Earn

Complete the challenges, earn back Sparks, and write guides to help the next builder.

Get started

Join the Community

Hack Clubbers hang out on our Slack. For GameForge join #gameforge, #gameforge-help and #gameforge-announcement, there you'll get updates, ask for help, and connect with other participants.

Before you start forging

Choose a game engine you want make sure it has a wakatime plugin and start tracking your code time using hackatime and lapse for artlogs. There are setup guides available on the hackatime site, also you can see the supported extensions but some might not be listed.
We track the time using hackatime as it allows us to track time accurately.

Make a Github repository for your game, make sure it's public.

Create project

After RSVPing go to 'My Projects' tab and create a new project, Choose a constraint from the random constraint generator and start Forging.

What counts as
a shipped game?

Playable build

A web build on itch.io, or downloadable builds for Windows, Mac, or Linux. Any Player around the world must be able to play it.

Fits the constraint

Your game must be built around the chosen constraint. The constraint should be central to the design, not an afterthought.

Should be fun

It should have a gameplay loop - it needs to feel like a game.

Open source

Your source code must be publicly available on GitHub.
